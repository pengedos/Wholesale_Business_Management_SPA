About

Built as a portfolio-ready wholesale e-commerce experience for frozen food operations.

This project demonstrates a front-facing catalog and quote workflow for a frozen food distributor, combining responsive UI design with business-system logic such as product data, localStorage persistence, search, filtering, wishlist, and quote-cart behavior, request-form validation, quote reference generation, locally saved quote/order history, and browser-based status tracking, an admin-style dashboard mockup for KPI review, pipeline monitoring, stock visibility, recent quote management, and a Phase 4A backend-ready data-service layer for future Supabase/database integration.

Workflow

Wholesale buying flow from catalog review to submitted quote request.

Buyers can browse SKUs, filter by category, build a quotation cart, complete a buyer-information form, receive a generated quote reference, and revisit saved quotation records in a local history panel. This keeps the journey closer to B2B procurement than a simple retail checkout.
